📱 Bedrock Edition

  1. Open Minecraft Bedrock Edition.
  2. Go to Play → Servers.
  3. Select Add Server.
  4. Use address play.wilden.fun and port 19132.
  5. Save and join.

💻 Java Edition

  1. Open Minecraft Java Edition.
  2. Go to Multiplayer.
  3. Click Add Server.
  4. Use play.wilden.fun.
  5. Join the server.

🏡 Land Claims

Land Claims protect your builds, items, and progress. Everything inside your claim is protected and only accessible to players you trust.

Getting Started

  • Hold a golden shovel.
  • Right-click a block for the first corner.
  • Right-click another block for the second corner.
  • Your land is now protected.

Claim Blocks

  • 125 claim blocks per hour of playtime.
  • 125 claim blocks per vote.
  • /buyclaimblocks costs $25 per block.
  • Claim blocks are refunded when resizing or deleting claims.

Commands

CommandUse
/claimsguideView claim help.
/buyclaimblocksBuy more claim blocks.
/abandonclaimDelete current claim.
/trust [player]Allow building access.
/untrust [player]Remove access.
/accesstrust [player]Allow doors/buttons only.
/containertrust [player]Allow chest access only.
/trustlistView trusted players.
/banfromclaim [player]Block a player from entering.
/unbanfromclaim [player]Allow a banned player again.
Length and Surge use this system to help keep gameplay fair and protected.

🌿 Branches

Branches is Wilden’s custom progression system. Choose a branch with unique passive perks and unlock upgrades as you progress.

🌊 Tides

Water-focused branch with faster swimming, water breathing, underwater visibility, and fishing luck.

🌿 Roots

Nature-focused branch with faster crop growth and ore chances in natural biomes.

🔥 Ember

Combat-focused branch with fire aura, fire resistance, and fire aspect chances.

🌾 Verdant

Farming and mining branch with crop bonuses and extra ore chances.

Use /branches switch to switch branches. The first switch is free. The second costs $100,000 and doubles each time. Upgrades are lost and non-refundable.

📈 AuraSkills

AuraSkills rewards players for playing naturally. Mining, farming, combat, and other actions give XP, levels, stats, and rewards.

Commands

  • /skills — Main skills menu.
  • /mining, /farming, /combat — Skill menus.
  • /stats — Stats if enabled.
  • /skillpoints — Manage stat points if enabled.

👑 Custom Events System

Wilden has a 48-hour event cycle where players use gems to influence good and bad events.

Good Events

75% chance. Contribute gems to extend them.

Bad Events

25% chance. Contribute gems to stop them early.

Wizard NPC

View timers, contribute gems, speed up countdowns, and check leaderboards.
